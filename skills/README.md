# Skills

Install a playbook skill with:

```bash
npx skills add wiseiodev/agentic-playbook/skills/<skill>
```

## Available Skills

### bootstrap

Interactive guide for hand-shaping a reference feature in a greenfield repo.

```bash
npx skills add wiseiodev/agentic-playbook/skills/bootstrap
```

### cleanup-worktree

Remove a merged slice's worktree and delete its branch.

```bash
npx skills add wiseiodev/agentic-playbook/skills/cleanup-worktree
```

### decompose

Break a PRD into ordered tracer-bullet slice specs.

```bash
npx skills add wiseiodev/agentic-playbook/skills/decompose
```

### implement

Run implementation against an approved plan.

```bash
npx skills add wiseiodev/agentic-playbook/skills/implement
```

### isolated-db-branches

Set up per-feature Neon database branches for Drizzle projects, with branch work using `db:push` and production migrations generated after merge.

```bash
npx skills add wiseiodev/agentic-playbook/skills/isolated-db-branches
```

### linearize

Turn an idea into a Linear Project-as-PRD and approved issue queue.

```bash
npx skills add wiseiodev/agentic-playbook/skills/linearize
```

### plan

Generate an implementation plan from a locked per-task spec.

```bash
npx skills add wiseiodev/agentic-playbook/skills/plan
```

### pr

Draft and open a PR after gates and review are clean.

```bash
npx skills add wiseiodev/agentic-playbook/skills/pr
```

### review

Run anti-overengineering review on the current diff.

```bash
npx skills add wiseiodev/agentic-playbook/skills/review
```

### setup-quality

Set up or tighten the repo's quality-gate toolchain.

```bash
npx skills add wiseiodev/agentic-playbook/skills/setup-quality
```

### spec

Create or open a per-task spec from the canonical template.

```bash
npx skills add wiseiodev/agentic-playbook/skills/spec
```

### work

Execute one scoped implementation slice end-to-end with the playbook's full trust loop.

```bash
npx skills add wiseiodev/agentic-playbook/skills/work
```

### worktree

Create a git worktree and branch for a slice.

```bash
npx skills add wiseiodev/agentic-playbook/skills/worktree
```
